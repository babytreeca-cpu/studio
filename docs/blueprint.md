# **App Name**: Report LJSM

## Core Features:

- Anonymous Incident Filing: Users can submit reports without accounts, selecting from multiple types of violence and providing incident details.
- AI Severity Triage Tool: Uses AI as a tool to summarize incident descriptions and suggest an urgency level to help administrators prioritize critical cases.
- Digital Evidence Secure Vault: Safe upload of images and documents to supplement reports, using a secure storage backend for later review.
- Unique Case Tracking System: Generates a unique tracking code via Cloud Functions, allowing anonymous users to check their case status safely.
- Role-Based Admin Dashboard: Authenticated environment for staff to manage reports, update case statuses, and view all evidence.
- Internal Case Documentation: Allows administrators to log observations and internal progress notes on specific cases stored in the database.
- Real-time Metrics Dashboard: A visual control panel for administrators showing total cases, status distributions, and categorical breakdowns.

## Style Guidelines:

- Primary color: Institutional Royal Blue (#1565C0), selected to convey trust, security, and institutional authority.
- Background: Soft Azure Mist (#F2F6FA), a heavily desaturated variant of the primary blue to provide a clean, high-contrast, professional canvas.
- Accent: Calm Sky Blue (#4FC3F7), used to create a peaceful contrast and guide users through form sections without visual stress.
- Font pairing: 'Inter' (sans-serif) for professional, machined headlines and labels; paired with 'PT Sans' (sans-serif) for body text to add human warmth and personality to sensitive descriptions.
- Cards and surface-driven layout using elegant shadows to separate administrative data from report content for clarity.
- Minimalist, thin-stroke icons that emphasize security and assistance, using the primary and accent blues to maintain cohesion.
- Discrete transitions when navigating from report submission to the confirmation view to instill a sense of calm and technical reliability.
